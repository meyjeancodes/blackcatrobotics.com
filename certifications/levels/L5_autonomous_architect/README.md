# L5 — Autonomous Systems Architect

The highest BCR field certification. L5 architects design fleet-scale AI systems, define new robot platform integrations, lead industry standards efforts, and develop the next generation of BCR engineers.

## Overview

| Field | Value |
|---|---|
| Level | L5 — Autonomous Systems Architect |
| Job Value | $2,500 – $4,500 / engagement |
| Prerequisites | L4 Systems Engineer + 3 years enterprise fleet experience |
| Assessment | 40 questions, 85% passing threshold |
| Time Limit | 120 minutes |
| Attempts | 3 (30-day cooldown) |

## Core Competencies

- Machine learning for predictive maintenance (LSTM autoencoders, Gradient Boosting, feature engineering, SHAP analysis)
- Edge AI deployment on NVIDIA Jetson AGX Thor (275 TOPS, TensorRT INT8, CUDA stream priority scheduling)
- BCR platform module authoring (signal ID mapping, diagnostic guide, common failures)
- Industry standards and compliance (IEC 61508, ISO 10218-1, ISO/TS 15066, EU AI Act, FAA Part 137)
- Industry leadership (standards contribution, open-source strategy, position paper authoring, mentoring)

## Curriculum Sections

1. Machine Learning for Predictive Maintenance — feature engineering, model architectures (LSTM autoencoder, Gradient Boosting, TCN), evaluation metrics (AUC-PR, precision/recall, calibration), distribution shift detection, federated learning
2. Edge AI — Jetson AGX Thor — 275 TOPS platform specs, TensorRT optimization pipeline (ONNX export, INT8 calibration, layer fusion), CUDA stream priority scheduling, thermal management
3. Platform Definition Authoring — BCR platform module standard (three required documents), canonical signal ID convention, signal metadata requirements, authoring workflow
4. Industry Standards and Compliance — IEC 61508 SIL levels, ISO 10218-1, ISO/TS 15066 contact force limits, EU AI Act high-risk requirements, agricultural drone regulations, data compliance frameworks
5. Industry Leadership — standards contribution methodology, open-source strategy, position paper authoring, A/B testing for ROI validation, mentoring L4 engineers

## Lab Exercises

- Lab 5-1: Predictive Maintenance ML Pipeline (XGBoost, SHAP, distribution shift monitoring)
