# L2 Technician Competency Rubric

| Rating | Label | Description |
|---|---|---|
| 4 | Expert | Performs without prompting, handles unexpected variations, could teach others |
| 3 | Competent | Performs correctly and consistently without assessor intervention |
| 2 | Developing | Completes task with minor prompting or 1-2 procedural errors |
| 1 | Insufficient | Cannot complete task or makes safety-critical errors |
