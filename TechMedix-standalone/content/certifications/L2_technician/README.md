# L2 Technician Certification

**Job Value Range:** $450 - $650 per job
**Prerequisites:** L1 Operator certification
**Written Exam:** 35 questions, 78% passing, 60 minutes
**Practical:** Required — assessor sign-off; prerequisite L1 practical must be current

## What L2 Unlocks

- Actuator disassembly and R&R (remove and replace)
- CAN bus capture and analysis using candump
- ROS 2 node health inspection and log analysis
- Oscilloscope and multimeter diagnostics at motor and bus level
- IMU calibration procedures
- Firmware update and post-repair validation
- AR-assisted diagnostics via TechMedix AR Mode

## Study Path

1. `curriculum.md` — five sections covering diagnostic tools through repair procedures

## Estimated Study Time

- Self-study: 35-50 hours
- Lab time: 16-20 hours (with assessor)
- Total: 50-70 hours after passing L1
